Code for Processing

To use Processing, for a static image, use noLoop() in your setup() method and then code your image in the draw() method. At the end of your draw, use saveFrame to save the display window image to a file.

For an animation, do not put noLoop() in your setup(). Instead, code your animation into the draw() function and keep track of the frame count somehow. When all the frames have been displayed, then call noLoop(). To create an animated GIF, follow one of the approaches described here. The examples below use the second suggested approach -- see the comments in the sketch code.
