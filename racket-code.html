Code for Racket

Using the student languages (BSL/ISL) of DrRacket, you can use the 2htdp/image library to create images of size 16x16 or 32x32. For additional functionality, download the save-gif.rkt file, save it to the same directory (folder) as your program code, and add the following expression to the top of your file:

(require "save-gif.rkt")

Saving PNG Images

To save a static (non-animated) PNG image, you can use the save-image function that comes with the 2htdp/image library.

Saving GIF Images

To save either a static or animated GIF, use the save-gif function that is provied by the save-gif.rkt file. It works like save-image for static GIFs, for example:

(save-gif IMG "test.gif")

To create an animation, provide a list of images instead of a single one. You may also specify a delay (in 1/100s of a second) as well:

(save-gif (list R S T) "testA.gif")
(save-gif (list R S T) "testB.gif" 20) ; 5 frames per second

Scaling Bitmap Images

To scale a bitmap image properly for submission for this contest, use the scale-bitmap function, e.g.:

(scale-bitmap 10 IMG)  ; scale IMG 10 times larger
